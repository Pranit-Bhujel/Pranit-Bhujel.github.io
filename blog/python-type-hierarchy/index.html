Multi-Line Statements and Strings¶

Understanding Multi-Line statements definitely makes us a better developer. Multi-Line statements/strings are generally used for improving the readability of the code.

First, we need to understand the difference between physical newline character and logical newline token.

Physical Newline Character: '\n' The character which is passed onto the program while we press enter to write a statement in the newline.

Logical Newline Token: Once we execute the program, the python compiler breaks the program into logical lines of code and they are tokenized. These tokens/lgical lines end with a logical newline token, which is then used by the python interpreter to execute the code.

Sometimes, physical newlines are ignored in order to combine multiple physical lines terminated by a logical newline token.

Now that we know what physical and logical newlines are, we can move forward to converting physical newlines to logical newlines.
